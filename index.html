Architecture

Built different.
From the kernel up.

Every component designed to extract maximum performance from CPU hardware that already exists in every machine on the planet.

// ISA_DISPATCH
Multi-ISA Runtime
One binary. Detects your CPU at startup and activates the fastest path — AVX-512 VNNI, AVX2+FMA, ARM NEON, or scalar fallback. Zero configuration.
// TERNARY_KERNEL
1.58-bit Quantization
Weights stored as {-1, 0, +1}. Multiplications become additions. 71% of weights are zero — skipped entirely. 16× smaller than FP32.
// PAGED_KV_CACHE
Paged KV-Cache
O(1) token insertion with block-based memory. Run 7B models on 8GB RAM via memory mapping. Only the active layer lives in memory.
// FUSED_KERNELS
Fused Operations
MatMul+RMSNorm and Ternary+SiLU run in single passes. 2–3× less memory traffic. Stays inside L1 cache where it matters.
// SHA256_VERIFY
SHA-256 Security
Kernel source verified against official hash before every compilation. Tampered code is detected and refused. Never compiled, never executed.
// NUMA_PINNING
NUMA Awareness
Detects multi-socket topology automatically. Pins threads to one NUMA node. Prevents the 30% throughput loss from cross-socket memory access.
